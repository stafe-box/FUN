<h5>Добавленны сокращения для вывода в консоль определённым текстом:
<h7>KNRM - Вернуть нормальный цвет;
KRED - Красный
KGRN - Зелёный
KYEL - Жёлтый
KBLU - Голубой
KMAG - Фиолетовый
KCYN - Синий
KWHT - Белый
Для объявления используется ключ %s. Пример:
  ```c 
  ("Input type of filling %s[1-by keyboard, 2-random]%s >>> ", KRED, KNRM);
  ```
